Theory

In this stage, you will improve your neural network. It will contain not only input and output layers, but also hidden layers.

The general representation of the multilayer neural network is shown in the picture below:

As you can see, it has a lot of layers. The one to the left is the input layer, and the one to the right is the output layer, and everything else is called hidden layers. Every layer can have a different number of neurons in it. But in practice, all hidden layers often have the same number of neurons.

Remember, that biases always have a value of 1 inside the neuron so there are no arrows to biases.

This specific neural network has $ r $ layers. The first layer has $ x $ input neurons and one bias, the second layer has $ n $ neurons and one bias, the third layer has $ m $ neurons and one bias, the pre-last layer has $ k $ neurons and one bias and the last layer has $ p $ output neurons.

Description

In this stage, you need to create such functionality that allows you to create neural networks with hidden layers and the ability to learn from a set of input data and the corresponding ideal output. The neural network should learn from 10 samples with 10 corresponding ideal outputs. Remember that ideal output is a set of numbers where there's just a single 1 and all others are zeros.

One thing that hasn't been discussed before is how to update weights in the middle when there is no ideal value for the neurons in the middle. You already know how to update weights from the next-to-last layer to the last layer since the last layer contains the ideal values for the neurons.

You can start from the end and define the ideal values for the neurons in the next-to-last layer. Just compute the value of the neuron in reverse order (from the last layer to the next-to-last one) using the same weights, but you should not multiply the value. You should divide the value of the output neuron to the corresponding weight. The ideal value of the neuron would be the mean of all the 10 input data, like in calculating $ \Delta w $ in the previous stage. Please do not rely on the number 10; it should be subject to change without rewriting the code. After that, you can repeat this process for the next-to-last layer and the layer before that, and so on to the input layer of the neural network. Then, you can update all the weights since all the neurons have the ideal value.

Small tip: break down the logic into separate classes and methods, then the problem becomes much easier to program.

By the way, the best way of storing weights, biases, and values of the neurons is by using matrices and calculating with matrix multiplication. You can check this link to understand it better and watch this to understand how to implement it in a neural network (this is the same video about neural networks). Since this project is not about matrix multiplication, you can use this implementation in your program.

Additionally, to get started with vectors and matrices, you can check out this series of videos. You only need the first 4 videos. The other videos dig deeper into linear algebra instead of vectors and matrices.
