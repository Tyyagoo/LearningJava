Repetitive Substring

Given a string $s$ and an integer $k$. Write a program that checks whether there is a substring of length $k$ that appears in $s$ at least twice.

Input: the first line contains a string $s$ such that $|s| \le 6 \cdot 10^6$. The second line contains an integer $k \le 500$.

Output: if $s$ contains a substring of length $k$ repeating at least twice, print this substring. Otherwise, output "does not contain".

In this problem, you are expected to implement an algorithm that utilizes string hashing and works in $O(|s|)$.


Sample Input:
abacabad
3

Sample Output:
aba


Sample Input:
abacabad
4

Sample Output:
does not contain



Memory limit: 256 MB
Time limit: 5 seconds
