The longest repetitive substring

Wow! This problem is kind of tricky. If you're ready to put your thinking cap on, brace yourself, and good luck! If, however, you find the problem too difficult, you may try to solve its simpler version first, and then return to this one.

Given a string \( s \). Write a program that finds a substring of the maximal length that appears in \( s \) at least twice.

Input: a string $s$.

Output: a maximum number $k$ such that there is a substring of length $k$ that appears in $s$ at least twice.

Try to come up with an algorithm that checks if there is a substring of length \( k \) that appears in \( s \) at least twice. Then, use binary search to find the maximal \( k \).


Sample Input:
ACTTGATTGA

Sample Output:
4


Sample Input:
ABCD

Sample Output:
0



Memory limit: 256 MB
Time limit: 8 seconds
