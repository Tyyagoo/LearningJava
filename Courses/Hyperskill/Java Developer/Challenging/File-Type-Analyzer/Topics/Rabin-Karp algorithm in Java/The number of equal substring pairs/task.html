The number of equal substring pairs

Wow! This problem is kind of tricky. If you're ready to put your thinking cap on, brace yourself, and good luck! If, however, you find the problem too difficult, you may try to solve its simpler version first, and then return to this one.

Given a string \( s \) and a set of pairs, where each element represents the starting and the ending position of a substring of \( s \). Write a program that counts the number of pairs of equal substrings.

Input: the first line contains a string \( s \). The second line contains an integer \(t\). Each of the following \(t\) lines contains 4 integers \( i, j, k \) and \(n\) separated by space, such that \(0 \le i \lt j \le |s|, 0 \le k \lt n \le |s|\).

Output: the number of pairs, such that \( s[i:j] \) is equal to \(s[k:n]\).

In this problem for a polynomial hash with constants \( a = 53 \) and \( m = 10^{9} + 9 \) it is guaranteed that if hash values for two substrings are equal, then the strings are equal as well.


Sample Input:
abacabad
2
1 4 5 8
0 3 4 7

Sample Output:
1


Sample Input:
seamlessness
3
4 8 8 12
5 8 9 12
0 1 11 12

Sample Output:
2
