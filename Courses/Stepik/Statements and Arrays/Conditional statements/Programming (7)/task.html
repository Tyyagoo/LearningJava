A chocolate bar has a rectangular shape, divided into NxM segments. You can break down this chocolate bar into two parts by a single straight line (only once). Determine if you can break off exactly K segments of chocolate. (Each segment is 1x1.)

Input data format

User inputs three integers: N, M, K.

Output data format

The program must output one of the two words: YES or NO.


Sample Input:
4
2
6

Sample Output:
YES



Memory limit: 256 MB
Time limit: 8 seconds
