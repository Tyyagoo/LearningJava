Write a program that reads the number of the direction (1 – up, 2 – down, 3 – left, 4 – right, 0 – stay) and outputs move up (or "move down", or "move left", or "move right", or "stay" depending on the entered number). If the number does not correspond to any of the listed directions, the program should output error!

Note: Please make sure your output exactly matches the required string, including capitals, spaces, and punctuation marks.


Sample Input:
1

Sample Output:
move up



Memory limit: 256 MB
Time limit: 8 seconds
