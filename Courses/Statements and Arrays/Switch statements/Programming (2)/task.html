A simple calculator

Write a simple calculator that reads three values from a line: the first number, the operator, and the second number, separated by single spaces.

The program should apply the operation to the entered numbers and output the numerical result to the screen. Note that the numbers are long.

The calculator should support the following operations / operators:

  • addition: "+"
  • subtraction: "-"
  • integer division: "/"
  • multiplication: "*"

If division is requested and the second number is 0, output Division by 0!

If no valid operator is present, output Unknown operator.

Note: Consider using the switch statement in your solution.


Sample Input:
10000000000 + 20000000000

Sample Output:
30000000000



Memory limit: 256 MB
Time limit: 8 seconds
