Given the sequence of natural numbers. For each number of the sequence output "even" if the number is even, otherwise - "odd". If the number is equal to 0, the program must stop reading and processing numbers.

Input data

A sequence of natural numbers, each number in a new line.

Output data

The sequence of words "even" and "odd". Each word in a new line.


Sample Input:
1
2
3
4
0

Sample Output:
odd
even
odd
even



Memory limit: 256 MB
Time limit: 8 seconds
