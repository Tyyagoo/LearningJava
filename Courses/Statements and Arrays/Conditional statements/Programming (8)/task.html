In a computer game, a player has an army of units. Write a program that will classify the player's army according to the following rules:

Total units: Category
fewer than 1: no army
from 1 to 4: few
from 5 to 9: several
from 10 to 19: pack
from 20 to 49: lots
from 50 to 99: horde
from 100 to 249: throng
from 250 to 499: swarm
from 500 to 999: zounds
1000 or more: legion

User inputs the number of units. The program outputs the corresponding category.

Sample Input:
5

Sample Output:
several



Memory limit: 256 MB
Time limit: 8 seconds
